IF USING ROS2 COMMANDS SEPARATELY
THERE IS 2 NODE THAT SHOULD BE RUN IN THIS ORDER
ros2 run take_image camera_run
ros2 run take_image ros2node

Alternatively for launch file:
ros2 launch filelocation/take_image_lauch.py

THERE IS 1 NODE THAT SHOUSLD BE RUN IN VISION IN THIS ORDER
ros2 run vision vision_main

ros2 launch filelocation/vision_launch.py
